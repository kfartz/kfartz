# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Kfartz.

## Global Shortcuts

These shortcuts work from anywhere in the application.

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` / `Cmd+K` | Open table switcher |
| `Ctrl+I` / `Cmd+I` | Go to insert page for current table |
| `Ctrl+L` / `Cmd+L` | Log out |

## Dialog Navigation

When dialogs are open (e.g., table switcher):

| Shortcut | Action |
|----------|--------|
| `Arrow Up` | Move selection up |
| `Arrow Down` | Move selection down |
| `Enter` | Confirm selection |
| `Esc` | Close dialog / Cancel |

## Table Switcher

The table switcher (`Ctrl+K` / `Cmd+K`) provides quick navigation between collections:

1. Press `Ctrl+K` / `Cmd+K` to open
2. Type to filter the list of tables
3. Use arrow keys to highlight
4. Press `Enter` to navigate to the selected table
5. Press `Esc` to cancel

## Tips

- On Mac, use `Cmd` instead of `Ctrl`
- Shortcuts are case-insensitive
- Shortcuts are disabled when typing in form fields
