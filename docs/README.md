# Kfartz User Manual

Kfartz is a crystallography measurement metadata database for storing, organizing, and querying data related to crystal structure analysis.

## Quick Start

1. [Log in](getting-started/logging-in.md) with your credentials
2. [Navigate](getting-started/navigation.md) to the table you need
3. [View](user-guide/viewing-data.md) and [search](user-guide/searching.md) existing data
4. [Add new records](user-guide/adding-records.md) as needed

## User Roles

**Regular Users** can:

- View all data
- Create new records
- Search and filter data

**Administrators** can additionally:

- Edit existing records
- Delete records
- Manage user accounts
- Create chamber types
