# User Management

User management is performed through the admin panel. Only administrators have access to these features.

## Accessing the Admin Panel

Navigate to `/admin` to access the Payload CMS admin panel. Log in with your administrator credentials.

![](../assets/admin-panel.avif)

## Viewing Users

1. In the admin panel, find the **Users** collection in the sidebar
2. Click to view the list of all user accounts
3. Click any user to view their details

## Creating a New User

1. Navigate to the Users collection
2. Click **Create New**
3. Fill in the required fields:
   - **Email** - The user's login email (must be unique)
   - **Password** - Initial password for the account
4. Optionally enable **Admin** to grant administrator privileges
5. Click **Save**

## Editing a User

### Changing a User's Password

1. Open the user's record
2. Enter a new password in the password field
3. Click **Save**

### Granting or Revoking Admin Access

1. Open the user's record
2. Toggle the **Admin** checkbox
3. Click **Save**

## Best Practices

- Grant admin access only to users who need it
- Regularly review the user list to remove inactive accounts
- Use strong, unique passwords for all accounts
