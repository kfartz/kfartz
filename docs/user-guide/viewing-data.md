# Viewing Data

Data in Kfartz is displayed in table views. Each collection (Crystals, Measurements, Processings, Refinements, Publications) has its own table.

> [!IMPORTANT]
> Due to caching, you have to refresh the page to view the current data if any changes were made.

## Table Layout

Each table displays:

- **Rows** - Individual records
- **Columns** - Data fields for each record
- **Header row** - Column names with visibility controls

## Column Visibility

By default, the first five columns are visible. To customize which columns appear:

1. Look for the column visibility controls in the table header
2. Toggle columns on or off as needed
3. Your selection persists while viewing that table

This is useful when tables have many columns and you want to focus on specific fields.

## Scrolling and Pagination

Tables use infinite scroll pagination:

- Scroll down to load more records automatically
- The table loads additional records as you reach the bottom
- No need to click through pages

## Viewing Related Records

Some columns display related records (e.g., a measurement's crystal). These appear as ID numbers you can then search for.
