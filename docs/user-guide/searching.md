# Searching

Kfartz provides a search bar for filtering records in the current table. The search supports a query syntax that allows precise filtering.

## Basic Search

Type in the search bar to filter records. The search applies to the current table only.

![](../assets/search.avif)

## Query Syntax

See the [Search Syntax Reference](../reference/search-syntax.md).

## Tips

- Field names are case-sensitive and match the column names
- Start with broad searches and add conditions to narrow results
- Use contains (`:`) when you're unsure of the exact value
