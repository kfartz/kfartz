# Navigation

## Interface Layout

The Kfartz interface consists of:

- **Main area** - Displays the current table view
- **Search bar** - Filter records in the current table
- **Navigation bar** - Home button and menu button

![](../assets/main-ui.avif)

## Switching Tables

Press `Ctrl+K` (or `Cmd+K` on Mac) to open the table switcher dialog:

1. Type to filter the table list
2. Use arrow keys to navigate
3. Press `Enter` to select
4. Press `Esc` to cancel

![](../assets/table-switcher.avif)

## Keyboard Shortcuts

Kfartz supports keyboard shortcuts for common actions:

| Shortcut        | Action                              |
| --------------- | ----------------------------------- |
| `Ctrl/Cmd + K`  | Open table switcher                 |
| `Ctrl/Cmd + I`  | Go to insert page for current table |
| `Ctrl/Cmd + L`  | Log out                             |
| `Arrow Up/Down` | Navigate lists and dialogs          |
| `Enter`         | Confirm selection                   |
| `Esc`           | Close dialogs                       |

See the full [Keyboard Shortcuts Reference](../reference/keyboard-shortcuts.md) for details.
